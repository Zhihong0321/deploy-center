Getting Started

Deploy-center monitors your Railway deployments and compares them with GitHub commits. Here's how to set it up:

  1. Go to the Repo Mapping tab
  2. Add each Railway service you want to monitor (you'll need the service ID from Railway dashboard URL)
  3. Come back here and click Refresh
  4. You'll see sync status for each service: IN_SYNC, BEHIND, or FAILED
